# Repositório de Imagens do PDTI

Este diretório reúne todos os recursos visuais utilizados no Plano Diretor de Tecnologia da Informação (PDTI) da Prefeitura de Hortolândia.  

Inclui:

- fluxogramas operacionais;  
- topologias de data center;  
- organogramas técnicos;  
- diagramas de processos;  
- ilustrações complementares aos documentos oficiais.

As imagens aqui armazenadas são utilizadas exclusivamente para fins de documentação, comunicação institucional e apoio visual às entregas do PDTI.
